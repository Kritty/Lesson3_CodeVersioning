Highlight #1 Versioning Control

Versioning control refers to the using of repositories in order to share files, make changes from multiple sources at multiple times, and saving them back in to one place that tracks any and all changes made to the documents and files saved to the repository.

Highlight #2 Versioning Control System

The VCS, or Versioning Control System allows for a common repository, where all the latest files can be held, and does not let files be overwritten. It also allows for people to work on the same file silmultaneously without conflict, and if desired, files may be reverted back to their original version when needed.

Highlight #3 GIT

GIT is the free and open VCS that is a source distributed system with the capabilities of handling everything from small to very large projects quickly and easily. GIT is also user-friendly and allows useers a complete sopy of the repository data stored locally on their machine.
